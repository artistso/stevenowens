# Chapter Drafts

No chapter is treated as final until Steven approves it.

## Chapter template

```markdown
# Chapter [number] — [working title]

Status: planned | drafted | fact-checked | author-approved
Timeline entries: [links or identifiers]
Source labels: T / R / P / D / X

## Scene

[First-person draft]

## Verification notes

- [ ] dates checked
- [ ] names checked
- [ ] sequence checked
- [ ] dialogue is remembered, recorded, paraphrased, or removed
- [ ] no new facts were added during editing
- [ ] Steven approved the public version
```

## Drafting order

1. Steven selects the event.
2. The event is entered in `../TIMELINE.md`.
3. Supporting material is entered in `../SOURCE_LEDGER.md`.
4. A plain factual draft is written.
5. Steven corrects the facts and wording.
6. Repetition is removed under `../VOICE.md`.
7. The approved chapter is committed here.

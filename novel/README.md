# Life Novel Project

This directory is the working area for a chronological first-person book based on Steven Owens's approved words, dates, projects, and public record.

The book will use direct language, verified events, and scene-based chapters. It will not add invented dialogue, filler, or unsupported claims.

## Files

- `OUTLINE.md` — chronological structure
- `TIMELINE.md` — dated event ledger
- `VOICE.md` — writing rules
- `SOURCE_LEDGER.md` — source tracking
- `HUGGING_FACE.md` — optional open-model workflow
- `chapters/` — approved chapter drafts

Chapter prose begins after the opening event is selected.

Book in progress

A chronological memoir-novel in Steven's own words.

The Life I Built is the provisional title of a first-person book built from dated events, approved testimony, repository history, public records, and Steven's corrections. Events are placed in a timeline before prose is written.

The book prohibits invented dialogue, invented motives, generic inspirational language, and unsupported facts. Hugging Face models may assist with chronology checks and line editing, but they cannot create memories.

Central argument

Not a loose project collection: a vernacular R&D studio.

The portfolio is most coherent when interpreted as a studio whose outputs move among software engineering, interface invention, speculative formalization, public testimony, humor, and civic critique. GitHub is not merely code storage. It is a laboratory notebook, publication channel, evidence ledger, and coordination surface for human–AI production.

Recurring commitments include local-first ownership, no-subscription creative software, stylus-centered interaction, circular or orbital interfaces, open publication, and AI agents used as inspectable engineering collaborators rather than hidden extractive features inside artwork.

Dissertation finding

The stable unit is not the individual repository. It is the authorship system connecting identity, iteration, evidence, distribution, and public explanation.

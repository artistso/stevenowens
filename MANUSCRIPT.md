# The SoQuarky System

**Vernacular Polymathy, Agentic Software Production, and Public Authorship Across Creative Technology, Speculative Science, and Social Media**

*An independent dissertation-style monograph by Steven Michael Allen Owens (@soquarky), Version 1.0, July 2026.*

> **Publication status:** This is an independent scholarly monograph and portfolio study. It is not represented as a degree awarded by an accredited university. First-person testimony is identified as testimony; repository assertions are identified as repository assertions; speculative scientific and medical proposals are not presented as externally validated merely because they include equations, simulations, code, tests, badges, or publication-style formatting.
